# Introduction

Welcome to the **@aptly/services** documentation. This library provides production-ready, lightweight backend services for the Aptly modular platform.

## What is @aptly/services?

It is a collection of modular TypeScript services designed for **high performance** and **zero-trust environment configuration**. Each service is isolated, strictly validated with Zod, and optimized for serverless and containerized deployments (like Coolify).

## Key Features

- 🧩 **Modular**: Selective imports keep your bundle sizes small.
- 🛡️ **Strictly Validated**: Runtime environment checks ensure your secrets are configured correctly.
- 🚀 **Lightweight**: No heavy dependencies (like Puppeteer or dedicated vendor SDKs where not needed).
- 🤖 **AI-Ready**: Designed with semantic structure and clear APIs for easy integration with AI coding agents.

## Quick Reference

| Service | Purpose |
| :--- | :--- |
| **Auth** | JWT & Bcrypt password hashing |
| **Email** | SMTP client with template support |
| **PDF** | Native, memory-efficient PDF generation |
| **SSO** | Google, Microsoft, and Zoho integration |
| **Storage** | S3-compatible file operations |
| **Payments** | Razorpay SDK wrapper |
| **WhatsApp** | Meta Cloud API notifications |
| **QR Code** | Stateless QR generation |
| **Cron** | Lightweight job scheduler |
